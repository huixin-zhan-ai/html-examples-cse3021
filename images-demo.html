Images in HTML

This page demonstrates <img> with src, alt, sizing via width/height and CSS, relative vs absolute URLs, and using an image as a link.

Accessibility tip: Always provide meaningful alt text describing the image’s purpose. If an image is purely decorative, you can use alt="".

Key properties

  • src — where the image comes from (relative or absolute URL).
  • alt — text shown to screen readers or if the image fails to load.
  • width/height — intrinsic size hints; use CSS for responsiveness.

This demo aligns with the slide’s points: <img> is an empty element (no closing tag) and commonly uses src, alt, and sizing attributes.
