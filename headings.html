HTML Headings Demo (h1–h6)

This page demonstrates semantic headings from <h1> to <h6>. Use headings for document structure—not just to make text big or bold.

Course Title (h1)

Top‑level page/topic title. There should generally be a single h1 per page.

Unit 1: Introduction (h2)

Major section of the page.

1.1 What is HTML? (h3)

Subsection inside Unit 1.

1.1.1 HyperText (h4)

Detail under the subsection.

Historical Note (h5)

Less prominent detail.

Footnote / Aside (h6)

Least prominent heading level.

Key points

  • <h1> is the most important heading; <h6> is the least.
  • Search engines use headings to understand page structure and content.
  • Use headings for headings only—don’t use them just to change font size or weight.

Try it

  1. Change the h1 text and refresh the browser to see the new top title.
  2. Add another h2 section with a couple of h3 subsections.
  3. Open DevTools (Right‑click → Inspect) to view the heading outline in the Elements panel.
